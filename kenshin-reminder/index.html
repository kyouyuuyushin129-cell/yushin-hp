従業員の健康診断予定をまとめて管理・LINE通知

🔔 通知対象の従業員

健診予定日まで 30 日以内の従業員です。（日数は「LINE設定」タブで変更できます）

⚠️ 未受診（期限超過）の従業員

健診予定日を過ぎても「受診済み」になっていない従業員です。受診が確認できたら「受診済み」を押してください。

📅 全員の予定

登録されている全従業員の健診予定です。

👤 従業員を追加

氏名と次回健診予定日を入力してください。部署・前回受診日・メモは任意です。

入力すると受診サイクルの管理に使われます

💬 LINE通知の設定

Google Apps Script（GAS）で作った送信用URL（Webhook）を登録すると、このアプリからワンタップでLINEに通知できます。URLが未設定でも「コピー」ボタンでメッセージを作成し、LINEに貼り付けて送れます。

下の「設定手順」に従って作成したURLを貼り付けてください
使える差し込み： {name} 氏名 ／ {date} 予定日 ／ {days} 残り日数 ／ {dept} 部署

🛠 設定手順（初回のみ）

LINE公式アカウント（Messaging API）+ Google Apps Script を使った無料構成です。

  1. LINE Developers（developers.line.biz）でMessaging APIチャネルを作成し、チャネルアクセストークン（長期）を発行する。
  2. 通知を受け取りたいLINEアカウント（またはグループ）に作成した公式アカウントを友だち追加し、ユーザーID / グループIDを控える。
  3. script.google.com で新規プロジェクトを作成し、下の「GASコード」を貼り付けて、トークンと送信先IDを書き換える。
  4. 「デプロイ → 新しいデプロイ → ウェブアプリ」で 全員がアクセス可能 にして公開し、発行されたURLを上の「送信用URL」に貼り付ける。
  5. 「テスト送信」でLINEに届けば設定完了。
📄 GASコードを表示（コピーして使用） 
const TOKEN = 'ここにチャネルアクセストークン';
const TO    = 'ここに送信先のユーザーID/グループID';

function doPost(e) {
  const body = JSON.parse(e.postData.contents);
  UrlFetchApp.fetch('https://api.line.me/v2/bot/message/push', {
    method: 'post',
    contentType: 'application/json',
    headers: { Authorization: 'Bearer ' + TOKEN },
    payload: JSON.stringify({
      to: TO,
      messages: [{ type: 'text', text: body.message }]
    })
  });
  return ContentService.createTextOutput('ok');
}
