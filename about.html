01 / Vision & Mission

私たちが目指す場所

VISIONビジョン

建築現場の根幹を担う専門集団として、
業界の信頼と発展に貢献する。

MISSIONミッション

施工現場における搬入・荷揚げを専門的に担うことで、
建築業界の生産性と信頼性の向上に寄与し続ける。

02 / Values

私たちのバリュー

4つのバリューが、優心の行動指針の核心です。

VALUE 01

誠実な現場主義

現場の仕事に手を抜かない。どんな状況でも、目の前の仕事に誠実であり続ける。

VALUE 02

専門性の継続的な追求

搬入・荷揚げのプロとして、技術と知識を磨き続ける。「なんとなく」で動かない。

VALUE 03

関わる人すべてへの敬意

一緒に働く仲間も、現場の取引先も、等しく尊重する。一人ひとりの可能性を信じ、向き合い続ける。

VALUE 04

利他の精神

仲間のために、職人さんのために、現場のために。誰かの「助かった」を生み出す行動こそが、優心の原動力。

MESSAGE

03 / Message

誰かの「助かった」のために。
それが、優心の原動力です。

「荷揚げ屋」は、建築現場で資材を上の階へと運ぶ、縁の下の力持ちの仕事です。 暑い・狭い・重い——決して楽な仕事ではありません。それでも私たちが胸を張れるのは、 この一歩一歩が、現場全体を前に進めているという誇りがあるからです。

優心が何より大切にしているのは "利他" の心。 仲間のために、職人さんのために、現場のために動く——その積み重ねが信頼になると信じています。 未経験から始めたスタッフも、今では立派に現場を支える一人です。

これからも、誠実に・安全に・前向きに。 関わるすべての人に「優心に頼んでよかった」と思っていただけるよう、現場とともに歩み続けます。
