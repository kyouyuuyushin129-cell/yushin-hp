会社概要

ホーム / 会社概要

Vision

ビジョン

建築現場の根幹を担う専門集団として、
業界の信頼と発展に貢献する。

Mission

ミッション

施工現場における搬入・荷上げを専門的に担うことで、
建築業界の生産性と信頼性の向上に寄与し続ける。

Values

私たちのバリュー

3つのバリューが、優心の行動指針の核心です。

VALUE 01

誠実な現場主義

現場の仕事に手を抜かない。どんな状況でも、目の前の仕事に誠実であり続ける。

VALUE 02

専門性の継続的な追求

搬入・荷上げのプロとして、技術と知識を磨き続ける。「なんとなく」で動かない。

VALUE 03

関わる人すべてへの敬意

一緒に働く仲間も、現場の取引先も、等しく尊重する。一人ひとりの可能性を信じ、向き合い続ける。
