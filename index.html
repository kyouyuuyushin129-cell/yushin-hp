Values

私たちが大切にしていること

優心が現場で体現する3つのバリュー。
これが私たちの仕事の根拠であり、原動力です。

VALUE 01

誠実な現場主義

現場の仕事に手を抜かない。
どんな状況でも、目の前の仕事に誠実であり続ける。

VALUE 02

専門性の継続的な追求

搬入・荷上げのプロとして、技術と知識を磨き続ける。
「なんとなく」で動かない。

VALUE 03

関わる人すべてへの敬意

一緒に働く仲間も、現場の取引先も、等しく尊重する。
一人ひとりの可能性を信じ、向き合い続ける。
