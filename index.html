About Us

建築現場の“根幹”を、
誇りを持って担う専門集団。

荷揚げ屋は、建築現場で資材を上の階へと運ぶ、縁の下の力持ち。 暑い・狭い・重い——決して楽ではない仕事だからこそ、私たちはその一歩一歩に誇りを持っています。 「誰かの助かった」を生み出す利他の心で、関西から東海まで、数多くの現場を支えています。

Values

私たちが大切にしていること

優心が現場で体現する4つのバリュー。これが私たちの仕事の根拠であり、原動力です。

VALUE 01

誠実な現場主義

現場の仕事に手を抜かない。どんな状況でも、目の前の仕事に誠実であり続ける。

VALUE 02

専門性の継続的な追求

搬入・荷揚げのプロとして、技術と知識を磨き続ける。「なんとなく」で動かない。

VALUE 03

関わる人すべてへの敬意

一緒に働く仲間も、現場の取引先も、等しく尊重する。一人ひとりの可能性を信じる。

VALUE 04

利他の精神

仲間のために、職人さんのために、現場のために。誰かの「助かった」を生み出す行動こそ原動力。
